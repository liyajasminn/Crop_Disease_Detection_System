Crop Disease Detection using Computer Vision

Overview:

This project uses Deep Learning and Computer Vision techniques to detect crop diseases from leaf images.
The system classifies plant diseases using CNN and ResNet50 Transfer Learning models and provides predictions through a FastAPI backend.

Features:

Image Classification using CNN

Transfer Learning with ResNet50

Data Augmentation

Confusion Matrix Generation

Classification Report

FastAPI Prediction API

Disease Prediction with Confidence Score

Dataset
